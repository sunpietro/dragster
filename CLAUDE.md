# CLAUDE.MD

This repository contains codebase for the drag'n'drop library that will work in every frontend environment - React, Angular, Svelte, native.

The library is exposed to the community via NPM page: https://www.npmjs.com/package/dragsterjs
The library is hosted on Github: https://github.com/sunpietro/dragster

In the package.json file there are defined scripts for running tests, releasing the package and serving the demo locally.
